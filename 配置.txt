# 项目文件结构说明

## CSS 架构 (ITCSS)
├── settings/
│   ├── _variables.css     # 全局变量
│   └── _breakpoints.css   # 响应式断点
├── tools/
│   └── _mixins.css        # 混合宏（如需使用预处理器）
├── generic/
│   ├── _reset.css         # 重置样式
│   └── _normalize.css     # 标准化样式
├── elements/
│   ├── _typography.css    # 排版
│   └── _links.css         # 链接样式
├── objects/
│   ├── _container.css     # 容器
│   ├── _grid.css          # 网格系统
│   └── _media.css         # 媒体对象
├── components/
│   ├── _buttons.css       # 按钮
│   ├── _cards.css         # 卡片
│   ├── _forms.css         # 表单
│   ├── _navigation.css    # 导航
│   └── _sections/         # 页面区块
│       ├── _hero.css
│       ├── _features.css
│       ├── _services.css
│       └── _testimonials.css
├── utilities/
│   ├── _spacing.css       # 间距
│   ├── _typography.css    # 文字工具类
│   ├── _colors.css        # 颜色工具类
│   └── _display.css       # 显示属性
└── main.css               # 主入口文件

## ITCSS 层级说明
1. 设置层 (Settings) - 变量定义
2. 工具层 (Tools) - 混合宏和函数
3. 基础层 (Generic) - 重置和标准化
4. 元素层 (Elements) - 无类型选择器样式
5. 对象层 (Objects) - 布局模式
6. 组件层 (Components) - UI组件
7. 工具类层 (Utilities) - 辅助类
